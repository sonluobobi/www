Class RequestDispatcherBase

Description
Description | Descendents | Vars (details) | Methods (details)

Implements interfaces:

  • IRequestDispatcher

IRequestDispatcher的抽象实现，它实现了dispatch方法，并且定义了getCtrlClassName和getCtrlMethodName两个抽象方法，其子类只需实现这两个方法即可。
